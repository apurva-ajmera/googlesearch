Arrays Data Structure

Array is a linear data structure.In array we can randomly access any value,you might think how is that possible?The reason behind  is the array stored in main memory in contigous manner.Array store homogeneous data meaans if you initialize array with integer data type it can store inly integer data.

In array,you can perform random access.The truth behind for random access is that array use base address and index pointer.Now we discuss what is base address and index pointer.Base address of an array is the address of first element address of an array in main memory.index pointer means at which position you want to access an element.

Let's take an example,you declare an array of size 10.You have to remember it that first index of an array is 0.When you initialize array of size 10 the range of array's index is 0-9.Initialization means you give the size of an array,in this example the size of an array is 10 and range 0-9 means array's  first position is 0 and last position is 9.Assume that each element of an array is 1 byte so total array size is 10 byte.For accessing an element you have to perform just one operation and this operation is given below.

Base address + Index pointer .

If you want to access an element at position 5 then take base address of an array and add index pointer.Assume that base address is 2000 and here index pointer is 4 because we want to access an element at position 5 and our starting index is 0 so we subtract 1 from its position.Note that we take position 5 not index if we want access an element at index 5 so we can not perform any subtraction operation.so now our result is 2000+4 = 2004.So, at 2004 memory location fifth postion of an array is present.In case of an index we just add the index and it's base address means if we want to find address of fifth index then the address is 2005(2000+5).

100 200 300 400 500 600 700 800 900 1000

As we can see in above example array starts at 2000 and last address of an array is 2009.Now we discuss how to initialize it,implement it and use it.
Initialization of an array :- int a[n];   //where n is the size of an array

Acceessing array elements:- cout<<a[5]; //acceessing an element at index 5.

Below is the example of one dimensional array.

code of an array:- 


#include<iostream>

using namespace std;



int main()

{

    int n,num;

    cout<<"what is the size of an array:";

    cin>>n;

    int a[n];

    for(int i=0;i<n;i++)

    {

         cin>>num; //take input from the user

         a[i] = num;

    }

    //print an array

    for(int j=0;j<n;j++)

    {

         cout<<a[j]<<" ";

    }

}

Multidimensional Array Data Structure

In previous article we have seen that what is one dimensional array.One dimentional array means you  have only one row but in real scenario you might need more than one dimension for array.So,in this article we see what is multidimensional array.

Multidimensional means you have more than one row or column.Means when you require mulitple rows or column you have to use multidimensional array.Let's take an example you want to represent a matrix so you need more than one column and row.Assume you want to represent 3 by 3 matrix so that you need 3 rows and 3 columns.Before jump into actual code we start from how to initialize multidimensional array. 


int a[r][c];  //r stands for row and c stands for column

int a[3][3];  //this array have 3 rows and 3 columns

int a[3][2];  //this array have 3 rows and 2 columns

As above we can see what initialization of multidimensional array looks like.Now we see how we can take input from the user and place int it's position.Here we need 3 by 3 matrix so  we need two for loops.You might think why we require two for loops.First for loop is for traversing in row and second for loop for traversing in column.Still you have confusion don't worry just see below figure: 


  0     1    2   //0,1,2 is row and column index

0[00   01   03]

1[10   11   12]

2[20   21   22]

Array starting from 0 so when we initialize it with for loop it start with 0 but it is not compulsory you can start from index 1 but in this case we start from index 0.So you can see that outside of rectangular bracket i write indexes of array and in bracket i write down its position.First position is 00 because at that place row oindex is zero and column index is zero now row index remain 0 becuase inner loop still executing therefore column index is increasing by one so next position is 01 the 02 and so on.Let's look at code:- 


int a[3][3];//initialization of an array

int num; //for taking an input



//first for loop for row

for(int i=0;i<3;i++)

{

 //inner for loop for column

    for(int j=0;j<3;j++)

    {

         cin>>num;

         a[i][j] = num;

    }

}

You think how it works so let's decode it:- 


first outer query run so,

for i=0:

     for j=0;{execute code} then j++ then check condition

     now j=1 and condition is j<3(condition is true)

   

     for j=1;{execute code} then j++ then check condition

     now j=2 and condition is j<3(condition is true)



     for j=2;{execute code} then j++ then check condition

     now j=3 and condition is j<3(condition is false)



now it is increment the outer loop variable which is i,so now i=1



for i=1:

     for j=0;{execute code} then j++ then check condition

     now j=1 and condition is j<3(condition is true)

   

     for j=1;{execute code} then j++ then check condition

     now j=2 and condition is j<3(condition is true)



     for j=2;{execute code} then j++ then check condition

     now j=3 and condition is j<3(condition is false)



now it is increment the outer loop variable which is i,so now i=2



for i=2:

     for j=0;{execute code} then j++ then check condition

     now j=1 and condition is j<3(condition is true)

   

     for j=1;{execute code} then j++ then check condition

     now j=2 and condition is j<3(condition is true)



     for j=2;{execute code} then j++ then check condition

     now j=3 and condition is j<3(condition is false)



now it is increment the value of i so now i=3 hence condition is false(i<3)



so now program outside of the for loop

That is the basic implementation of multidimensional array.We see in first part that how we can initialize the array,in second part we see how we can use array to represent a matrix,and now this is time to implement the whole code.

Below is the implementation of an array:- 


#include<iostream>

using namespace std;



int main()

{

  int a[3][3];  //intializtion of matirx

  int num;



  for(int i=0;i<3;i++)

  {

    for(int j=0;j<3;j++)

    {

      cin>>num;

      a[i][j] = num;

    }

  }



  //print matrix

  for(int i=0;i<3;i++)

  {

    for(int j=0;j<3;j++)

    {

      cout<<a[i][j]<<" ";

    }

    cout<<endl;

  }

}
